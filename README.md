# NeuralViral AI - Content Architect

NeuralViral AI is a cutting-edge platform designed to transform raw audio and video transcripts into viral social media content using advanced AI. Whether you're a YouTuber, Facebook creator, or TikToker, NeuralViral helps you craft the perfect hooks, titles, descriptions, and tags to maximize your reach.

## 🚀 Features

- **Audio/Video Transcription**: Seamlessly upload files or provide links to get high-accuracy transcripts.
- **AI-Powered Viral Content**: Automatically generates optimized content for YouTube, Facebook, and more.
- **SEO & Growth Hacks**: Get specialized SEO titles, viral hooks, and interaction tags.
- **Meta Tags & Hashtags**: One-click copy for platform-specific hashtags.
- **Export to PDF**: Generate professional reports of your viral content strategy.
- **Sleek Dark Interface**: A premium, responsive UI designed for modern creators.

## 🛠 Tech Stack

- **Frontend**: React 18, Vite, TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion (motion/react)
- **AI Engine**: Google Gemini AI (@google/genai)
- **Icons**: Lucide React
- **Export**: jsPDF, html2canvas

## 📦 Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/neural-viral-ai.git
   cd neural-viral-ai
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up Environment Variables**:
   Create a `.env` file in the root directory and add your Gemini API Key:
   ```env
   VITE_GEMINI_API_KEY=your_gemini_api_key_here
   ```
   *Note: You can get your free API key from [Google AI Studio](https://aistudio.google.com/).*

4. **Start the development server**:
   ```bash
   npm run dev
   ```

## 🏗 Building for Production

To create a production-ready build, run:
```bash
npm run build
```
The optimized files will be generated in the `dist` folder.
